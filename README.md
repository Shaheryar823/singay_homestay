Flask + PostgreSQL)
A complete hotel booking platform built for **Singay Homestay, Skardu**.
Features include real-time availability, room selection, billing summary, mobile-friendly UI, and
smooth user experience with TailwindCSS.
---
#■ Features
- Room listings with dynamic pricing (per night & whole stay)
- Real-time availability calculation
- Add/remove rooms with live total update
- Mobile billing drawer with toggle
- Rate calculator popup
- Fully responsive Tailwind design
- PostgreSQL + Flask backend
- Clean, optimized JavaScript logic
---
#■ Tech Stack
- **Backend:** Python, Flask, PostgreSQL
- **Frontend:** Jinja2, TailwindCSS, Vanilla JS
- **Deployment:** Railway (DB + Backend)
- **Tools:** GitHub, VS Code, Postman
